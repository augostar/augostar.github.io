Webpack是用于为浏览器构建JavaScript模块脚本的前端工具

全局安装 npm install -g webpack

安装到项目内 npm install –save-dev webpack

安装webpack脚手架 npm install -D webpack webpack-cli

安装webpack服务器 npm install -g webpack-dev-server

安装loader npm install -D style-loader css-loader
